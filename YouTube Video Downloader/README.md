# YouTube Video Downloader (Python)

A simple desktop application built with **Tkinter** that allows you to download YouTube videos using **yt-dlp**.
The app provides a clean graphical interface where you can paste a video URL and optionally choose a custom filename.

---

## Features

* Download YouTube videos in best available quality
* Automatic merging of video and audio into **MP4** format
* Optional custom filename support
* Simple and minimal GUI using Tkinter
* Error handling for invalid URL or filename

---

## Requirements

Make sure Python is installed, then install the required library:

```bash
pip install yt-dlp
```

Tkinter usually comes pre-installed with Python.

## How to Use

1. Paste a valid **YouTube video URL** in the input field.
2. (Optional) Enter a filename ending with `.mp4`.
3. Click **Download**.
4. The video will be saved in the same folder as the script.

---

## Notes

* If no filename is provided, the video title is used automatically.
* Internet connection is required for downloading.
* This tool is intended for personal and educational use only.
