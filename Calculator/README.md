# Calculator App (Python + Tkinter)

This is a simple desktop calculator built using Python and Tkinter.  
The goal of this project is to provide a clean and easy-to-use calculator interface while practicing GUI development in Python.

---

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Percentage calculation support
- Clear (`C`) and backspace (`←`) functionality
- Keyboard support (press **Enter** to evaluate)
- Dark-themed user interface
- Handles invalid input safely

---

## Technologies Used

- Python  
- Tkinter (built-in Python GUI library)

---

## Usage

- Click the buttons to enter numbers and operators
- Press **Enter** to evaluate the expression
- Use `C` to clear the input
- Use `←` to remove the last character

---

## Notes

- This application uses Python’s `eval()` function to evaluate expressions
- Designed mainly for learning and personal use
- Best tested on Windows systems

---

## Possible Improvements

- Better error feedback for invalid expressions
- Keyboard key highlighting
- Additional scientific calculator features

---

## How to Run the Application

1. Make sure Python is installed on your system.
2. Also install the DS-DIGITAL font for the calculator.
