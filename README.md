# Python Apps

This repository contains a collection of Python applications created for learning, practice, and experimentation.  
Each project is organized into its own folder and can be run independently.

The focus of this repository is on writing clean Python code, exploring different concepts, and building small but complete applications.

---

## Repository Structure

Each project folder typically contains:
- Python source files
- A project-specific README
- Any required assets or resources

---

## How to Run a Project

1. Make sure Python 3.x is installed.
2. Open the folder of the project you want to run.
3. Install required dependencies (if any).
4. Run the main Python file: python app.py
