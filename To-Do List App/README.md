# Python To-Do List App

This is a desktop To-Do List application built using Python and Tkinter.  
It allows users to create, manage, save, and load tasks through a simple and user-friendly graphical interface.

The app is designed for practicing GUI development, file handling, and event-driven programming in Python.

---

## Features

- Add new tasks
- Delete selected tasks
- Mark tasks as completed
- Unmark completed tasks
- Save tasks to a text file
- Open previously saved task files
- Clear all tasks with confirmation
- Keyboard shortcuts for faster usage
- Scrollable task list

---

## Technologies Used

- Python
- Tkinter (GUI)
- OS module (file handling)

---

## Project Structure


todo-app/
├─ app.py
├─ README.md
└─ Tasks/
  
> The `Tasks` folder is automatically created when saving tasks.

---

## How to Run the Application

### 1. Requirements
- Python 3.x installed on your system

### 2. Run the App
Open a terminal in the project folder and run:
python app.py

---

## Keyboard Shortcuts

| Action | Shortcut |
|------|---------|
| Add task | Enter |
| Delete selected tasks | Delete |
| Save tasks | Ctrl + S |
| Open tasks file | Ctrl + O |
| Clear all tasks | Ctrl + Delete |
| Mark task as done | Ctrl + M |
| Unmark task | Ctrl + U |

---

## Usage Notes

- Multiple tasks can be selected at once
- Completed tasks are marked with a check symbol (`✔`)
- Tasks are saved as `.txt` files
- Error messages are shown for invalid actions
